# hybrid-pages-12-3-1
Nextjs v12.3.1 project example using edge+node runtims and pages format (app dir not available in v12.x).

This example contains:
* Index page;
* Edge API Route;
* API Route;
* SSR page;
* SSR Node page;
* Middleware with redirect;
* Dynamic Route;
* Node Dynamic Route;
* Nested route;
